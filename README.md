# KamiNoMi - Japanese Washi Paper Art

Luxury minimalist website for handcrafted Japanese washi paper products inspired by authentic artisanal traditions.

## Setup

### Requirements
- Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to project directory
cd KamiNoMi

# Install dependencies
npm i

# Start development server
npm run dev
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Build for production:
```sh
npm run build
```

Deploy the `dist` folder to your hosting provider.
